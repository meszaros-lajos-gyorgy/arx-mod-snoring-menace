------------------------------------
 Snoring Menace mod for Arx Fatalis
------------------------------------

A simple Arx Fatalis mod that boosts all in-game snoring sounds to be more irritating

--------------
 Installation
--------------

Unpack the contents of this zip and add it into
your Arx Fatalis/Arx Libertatis game's root folder

-----------
 Uninstall
-----------

Remove the following files from your Arx Fatalis/Arx Libertatis game's root folder:
- speech/english/guard_sleep_1_snoar.wav
- speech/english/guard_sleep_1_snoar2.wav
- speech/english/guard_sleep_1_snoar3.wav
- speech/english/guard_sleep_1_snoar4.wav
- speech/english/guard_sleep_1_snoar5.wav
- speech/deutsch/guard_sleep_1_snoar.wav
- speech/deutsch/guard_sleep_1_snoar2.wav
- speech/deutsch/guard_sleep_1_snoar3.wav
- speech/deutsch/guard_sleep_1_snoar4.wav
- speech/deutsch/guard_sleep_1_snoar5.wav
- speech/francais/guard_sleep_1_snoar.wav
- speech/francais/guard_sleep_1_snoar2.wav
- speech/francais/guard_sleep_1_snoar3.wav
- speech/francais/guard_sleep_1_snoar4.wav
- speech/francais/guard_sleep_1_snoar5.wav

Or if you don't have any other mods installed, then just delete the "speech" folder
